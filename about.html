why does this site look like that?
  → 
I believe that a portfolio site should reflect a developer's personal taste in ui design at the moment. Currently, I've really enjoyed the raw and unfiltered web design of the early 90s, which I've tried to replicate here.
  → 
The web is founded on this vanilla HTML and CSS webpage generation. It means that this style is functional and efficient, while staying true to the original use of the web: text, hyperlinks, and the occasional image.
  → 
the internet is so hyperstimulating these days, and i wanted a break. there's beauty in monotony too.
  → 
I spent a long time debating redesigning my site in this style, mainly because someone might misinterpret this as unintentional and simply a lack of css knowledge. For the reasons listed above, I decided to go ahead with this style, despite the possible consequences.
